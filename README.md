# Homebrew Tap for Kizaki

Official Homebrew formulae for [Kizaki](https://kizaki.ai).

## Install

```bash
brew install kizakicorp/tap/kizaki
```

## Upgrade

```bash
brew upgrade kizaki
```

## Uninstall

```bash
brew uninstall kizaki
```

## About

This tap is updated automatically on every CLI release. Formulae are generated — do not edit them by hand.
